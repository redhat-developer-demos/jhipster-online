More details

Maven is the default build tool (Gradle is available on the classic generator).
Vue is the default front-end on this page; choose Quarkus in Backend framework for Quarkus + Vue.
MariaDB remains a common default; PostgreSQL and MongoDB presets are included in generated repos — see the devfile commands.
The DevSpaces devfile.yaml template is copied into the generated repository (token-replaced).
Tekton pipelines are generated under helm/templates/ (no standalone pipeline.yaml at repo root).
An in-repo helm/ chart and argocd/application.yaml are generated for OpenShift.
