Please note that the current version of this form only supports the following selections within JHipster Archetype:

  • Application Type: Monolith
  • FrontEnd: VueJS
  • BackEnd: Spring Boot Application
  • Database: MariaDB

Services, Databases and selections not listed above are currently not supported.

OpenShift Deployment

The generated repository includes a helm/ chart and argocd/application.yaml. Push the repository first; Fabric8 renders templates server-side, Argo CD uses Helm in-cluster.

{{ deployStatus }}

More details

Maven will be the build tool used for the project.
Angular will be the default front-end framework used for the project.
MariaDB will default to passwordless database connections as it's more secure option.
The DevSpaces devfile.yaml will be added.
The Tekton Pipeline jhipster-pipeline.yaml will be added.
An in-repo helm/ chart and argocd/application.yaml are generated for OpenShift (legacy template.yaml is deprecated).
